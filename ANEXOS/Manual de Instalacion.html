¿Qué es COA-Report?

COA-Report es un sistema de gestión integral de residuos peligrosos que permite:

  • ✅ Registrar y gestionar empresas, plantas y encargados
  • ✅ Crear manifiestos de residuos
  • ✅ Generar reportes automáticos
  • ✅ Exportar datos en múltiples formatos
  • ✅ Respaldar bases de datos
✅ Buena noticia: El instalador incluye todo lo necesario. ¡No necesitas instalar Java!

🪟 Instalación en Windows

Paso 1: Descargar el Instalador

Ve a la sección Descargas y descarga el archivo COA-Report-1.0.0.msi

Paso 2: Ejecutar el Instalador

Haz doble clic en el archivo COA-Report-1.0.0.msi

Si ves un aviso de seguridad, haz clic en "Ejecutar" para continuar

Paso 3: Asistente de Instalación

Se abrirá el asistente de instalación. Sigue estos pasos:

  1. Lee y acepta la licencia (marca "I accept")
  2. Selecciona la carpeta de instalación (por defecto: C:\Program Files\COA-Report\)
  3. Haz clic en "Instalar"
  4. Espera a que termine (puede tomar 2-3 minutos)

Paso 4: Finalizar

Cuando termine, marca "Crear acceso directo en el Escritorio" (opcional)

Haz clic en "Finalizar"

Paso 5: Iniciar la Aplicación

Busca "COA-Report" en el menú Inicio o haz doble clic en el acceso directo del escritorio

¡La aplicación está lista para usar!

Desinstalación

Para desinstalar COA-Report:

  1. Abre "Agregar o quitar programas" (desde Configuración)
  2. Busca "COA-Report"
  3. Haz clic en "Desinstalar"
  4. Confirma la acción

La base de datos se conservará en: %APPDATA%\COA-Report\

🐧 Instalación en Linux

Requisitos Previos

# Instalar Java 21 (Ubuntu/Debian)
sudo apt-get update
sudo apt-get install openjdk-21-jdk

# Verificar instalación
java -version

Instalación del Instalador

Opción A: Desde AppImage (recomendado)

chmod +x COA-Report-1.0.0.AppImage
./COA-Report-1.0.0.AppImage

Opción B: Desde ZIP portátil

unzip COA-Report-1.0.0.zip
cd COA-Report
./COA-Report.sh

Opción C: Desde compilación

git clone <repositorio>
cd COA-Report-App
mvn clean package
bash generate-exe.sh

Ubicación de Datos

La base de datos se guarda en:

~/.coa-report/coareport.db

Crear Acceso Directo en Menú

sudo nano /usr/share/applications/coa-report.desktop

# Copiar y pegar lo siguiente:
[Desktop Entry]
Version=1.0
Type=Application
Name=COA-Report
Exec=/home/usuario/COA-Report/COA-Report.sh
Icon=coa-report
Categories=Application;
Terminal=false

🍎 Instalación en macOS

Requisitos Previos

# Instalar Homebrew (si no lo tienes)
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Instalar Java 21
brew install openjdk@21

# Verificar instalación
java -version

Instalación

Opción A: Desde DMG (recomendado)

  1. Abre el archivo COA-Report-1.0.0.dmg
  2. Arrastra la aplicación a la carpeta Applications
  3. Abre Applications y haz doble clic en COA-Report
  4. Si ves alerta de seguridad, ve a Preferencias de Seguridad y permite la ejecución

Opción B: Desde ZIP portátil

unzip COA-Report-1.0.0.zip
cd COA-Report
./COA-Report.app/Contents/MacOS/COA-Report

Ubicación de Datos

La base de datos se guarda en:

~/Library/Application Support/COA-Report/coareport.db
ℹ️ Nota: Si tienes un Mac con Apple Silicon (M1/M2), la aplicación se ejecutará sin problemas con emulación Rosetta.

⚙️ Configuración

Windows C:\Users\[Usuario]\AppData\Roaming\COA-Report\

macOS ~/Library/Application Support/COA-Report/

Base de Datos

Respaldar Base de Datos

Desde el menú Archivo → Respaldar BD

Se crea un respaldo en: backups/coareport_YYYYMMDD_HHMMSS.db

Restaurar Datos

  1. Cierra COA-Report
  2. Navega a la carpeta de datos
  3. Elimina coareport.db
  4. Copia el respaldo renombrándolo como coareport.db
  5. Abre COA-Report nuevamente

Cambiar Contraseña

En la interfaz principal, ve a Archivo → Cambiar Contraseña (si está disponible)

Exportar Datos

Desde la pestaña Reportes:

  • Genera reportes en Excel
  • Exporta a CSV
  • Crea reportes en PDF (próximas versiones)

❓ Preguntas Frecuentes

¿Necesito Java instalado para usar COA-Report? ▼

No. El instalador .MSI incluye Java automáticamente. Solo descarga y ejecuta el instalador.

¿Dónde se guardan mis datos? ▼

Los datos se guardan en una carpeta específica según tu sistema operativo (ver sección Configuración).

Puedes respaldar desde el menú Archivo → Respaldar BD

¿Puedo instalar COA-Report en múltiples computadoras? ▼

Sí. El instalador puede instalarse en múltiples computadoras. Cada instalación tiene su propia base de datos.

Para compartir datos, usa la función de respaldo y restauración.

¿Qué navegadores soporta la aplicación? ▼

COA-Report es una aplicación de escritorio (no web), así que no requiere navegador.

Se ejecuta directamente en tu sistema operativo.

¿Cuánto espacio en disco requiere? ▼

La instalación requiere ~500 MB. La base de datos crece según los datos que registres.

Típicamente, unos miles de registros ocupan menos de 10 MB.

¿Es segura mi información? ▼

Los datos se almacenan localmente en tu computadora con SQLite.

Realiza respaldos periódicos desde Archivo → Respaldar BD

¿Hay versión web o móvil? ▼

Actualmente solo existe la versión de escritorio (Windows, Linux, macOS).

Las versiones web y móvil están en desarrollo para futuras releases.

🔧 Solución de Problemas

La aplicación no inicia

Error: "El programa no pudo iniciarse"

Soluciones:

  1. Reinicia tu computadora
  2. Desinstala y reinstala la aplicación
  3. Verifica que tengas espacio en disco suficiente
  4. En Windows, ejecuta como Administrador

Error de base de datos

Error: "No se pudo conectar a la base de datos"

Soluciones:

  1. Verifica que tienes permiso de lectura/escritura en la carpeta de datos
  2. Comprueba que no hay otra instancia de COA-Report ejecutándose
  3. Elimina el archivo coareport.db para que se reinicialice
  4. Restaura desde un respaldo anterior

Contraseña olvidada

⚠️ Solución:

  1. Cierra la aplicación
  2. Elimina el archivo coareport.db de la carpeta de datos
  3. Abre COA-Report nuevamente (se recreará la BD)
  4. Usa las credenciales por defecto: admin / admin123

Problemas de rendimiento

Si la aplicación va lenta:

  1. Comprueba que tienes suficiente RAM disponible
  2. Cierra otras aplicaciones
  3. Limpia el directorio de respaldos innecesarios
  4. Considera ejecutar desde SSD en lugar de HDD

Exportar reportes no funciona

Soluciones:

  1. Verifica que tienes datos en la tabla correspondiente
  2. Comprueba que tienes permisos de escritura en la carpeta de descargas
  3. Intenta guardar en otra carpeta
  4. Verifica que tienes espacio en disco suficiente
